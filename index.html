CAS Server Lab

简介

欢迎访问CAS Server Lab，旨在解决集成SSO中常用的场景分析及技术解决方案；

环境

tomcat8 + jdk 8
base on Cas 4.0.0

v0.0.0.1

  • [x] CAS服务端演示

    1.登录地址：https://localhost:8443/cas/login （admin/admin）

v0.0.0.2

  • [x] 支持数据库连接，从DB中验证密码

    1.DB连接池： HikariCP
    2.加密算法：Spring Security的BCrypt

v0.0.0.3

  • [x] 集成RESTful API
参考文献 https://wiki.jasig.org/display/CASUM/RESTful+API
